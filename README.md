# 🎥 YouTube Clone

A responsive YouTube homepage clone built using **HTML** and **CSS**. This project was created as part of my frontend development journey to strengthen my understanding of modern web layouts, Flexbox, CSS Grid, positioning, and responsive design.

---

## 📸 Preview

![YouTube Clone Preview](images/preview.png)



---

## 🚀 Features

- ✅ Responsive YouTube-inspired layout
- ✅ Fixed navigation header
- ✅ Responsive video grid
- ✅ Sidebar with navigation icons
- ✅ Search bar with hover tooltips
- ✅ Voice search button
- ✅ Notification badge
- ✅ User profile section
- ✅ Custom channel images and thumbnails
- ✅ Clean and organized folder structure

---

## 🛠️ Built With

- HTML5
- CSS3
- Flexbox
- CSS Grid

## 📚 What I Learned

Through this project, I gained hands-on experience with:

- Structuring web pages using semantic HTML
- Building responsive layouts using Flexbox and CSS Grid
- Creating fixed headers and sidebars
- Working with SVG icons and image assets
- Using hover effects and CSS transitions
- Positioning elements with `position: absolute` and `position: relative`
- Organizing CSS into multiple files for better maintainability

## 💻 Getting Started

Clone the repository

```bash
git clone https://github.com/The-Tech_Guy005/Youtube-Clone.git
```

Open the project

```bash
cd Youtube-Clone
```

Open `youtube.html` in your browser.
